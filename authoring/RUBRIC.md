# Quality rubric

Every generated item must pass all applicable criteria. The rubric reviewer
scores each item pass/fail per criterion with a one-line reason for any fail.

| # | Criterion | Applies to |
| --- | --- | --- |
| R1 | Answer key is correct (independently derivable) | questions |
| R2 | Exactly one defensible answer; each distractor is wrong for a nameable reason | mcq |
| R3 | Difficulty matches the 1–5 scale in STYLE.md | questions |
| R4 | Pitfall names a real misconception, not a restatement | questions |
| R5 | Hint ladder escalates nudge → approach → near-answer, never spoils | questions with hints |
| R6 | Not a duplicate of an existing question (same fact tested from the same angle) | questions |
| R7 | Why-first voice; backstory is concrete pain, not marketing | prose |
| R8 | Terminology consistent with the concept file and pack | all |

A finding = item id + criterion + one-line reason. No finding, no comment —
the reviewer reports failures only.
